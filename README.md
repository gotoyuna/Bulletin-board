<h1>Bulletin board（掲示板アプリ）</h1>

メンバー登録し、ログインすると自由にtitleとtextを入力して、つぶやきを投稿することができます。<br>
また、どのようなメンバーが登録されているか、Profileページを見ることができます。本人は、プロフィール画像やメッセージを編集することができます。投稿は本人であれば自由に削除、編集することができます。

<h2>使い方</h2>
<h3>【メンバー登録して、投稿したいとき】</h3>
①掲示板はログインしなくても、見ることができます。<br>
②メンバー登録して、ログイン（mailとpasswordを入力）すると掲示板につぶやきを投稿することができます。<br>
③パスワードは8〜15文字以内の半角英数字で入力しなければ、エラー文が表示されます。<br>
④ログインしていない人は、「Let's new post!」のリンクを押してもログイン画面が表示されます。<br>
⑤掲示板一覧には、投稿した人のNo（何番目か）、Name、Title、Text、Datetime、Operations（自分の投稿だけ、update、deleteが表示される）が投稿されます。<br>
⑥すでに会員の方は、ログインすると投稿することができます。<br>
⑦自分の投稿を編集(update)、削除(delete)できます。<br>
⑧Name欄にある名前を押すと、投稿した人のPrifileページを見ることができます。<br>
⑨ログインした人は、自分のProfileページを編集できます。<br>
⑩ログアウトを押すと、ログアウト完了ページが表示され、掲示板一覧に戻ることができます。<br>

<h3>【パスワードを忘れてしまったとき】</h3>
①掲示板一覧画面の「Login」または「Let's new post!」をクリックし、ログイン画面へ<br>
②「If you forgot your password, click here」をクリックし、Password reissue screen(パスワード再発行画面)へ<br>
③メンバー登録した際のメールアドレスを入力し、sendボタンをクリック<br>
④入力したメールアドレス宛に発行から30分間のみ有効かつ、そのURLからの変更は1回のみ可能な再発行用URLが送信される<br>
⑤ユーザーは入力したメールアドレス宛に届いたメール本文のURLをクリック<br>
⑥パスワード再発行画面にアクセスされ、そこで新規のパスワードを8〜15文字以内の半角英数字で入力<br>
⑦パスワード更新完了<br>
